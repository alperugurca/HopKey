# Privacy Policy — HopKey

**Last updated: March 2026**

## Overview

HopKey is a Chrome extension that redirects your browser to a target URL when you type a shortcut key in the address bar.

**HopKey does not collect, store, transmit, or share any personal data.**

## Data we do NOT collect

- No browsing history
- No search queries
- No personal information (name, email, location, etc.)
- No usage analytics or telemetry
- No financial or health information
- No authentication credentials

## What is stored locally

HopKey stores your shortcut key-to-URL mappings using Chrome's built-in `chrome.storage.local` API. This data:

- Never leaves your device
- Is never sent to any server
- Is only used to perform redirects within your browser

## External requests

When you click "Import from URL" or "↓ Community", HopKey fetches a `shortcuts.json` file from a public GitHub URL. This is a one-way download of a plain JSON file. No data about you is sent to GitHub or any other server during this request.

## Third parties

HopKey does not use any third-party analytics, advertising, or tracking services.
